## 🎓 Metodologia de Estudo

### Para Cada Guia:

1. **Leia a teoria** - Entenda os conceitos
2. **Execute os exemplos** - Copie e rode no seu computador
3. **Modifique os exemplos** - Teste variações
4. **Faça os exercícios** - Pratique sem olhar a solução
5. **Confira a solução** - Compare com sua abordagem
6. **Crie algo próprio** - Aplique o que aprendeu

### Dicas de Estudo:

- ⏰ **Estude em blocos:** 25-50 minutos com pausas
- 📝 **Anote dúvidas:** Volte depois ou pergunte
- 💻 **Pratique diariamente:** 1 hora/dia > 7 horas no sábado
- 🎯 **Foque em entender:** Não decore, compreenda
- 🔁 **Revise:** Releia guias anteriores periodicamente
